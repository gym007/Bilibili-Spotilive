# Bilibili-Spotilive
B站直播弹幕点播Spotiy

# 如何使用

- 下载Bilibili-Spotilive-v1.0.2.zip
- 解压后打开config.json
- 获取对应bilibili网页cookie
- 输入主播id和房间号（你自己的）
- 申请Spotify开发者api
- 输入Spotify默认歌单链接
- 运行v1.0.2.exe

# 点歌指令
- 点歌 歌名 歌手(可选)
  - 点亮粉丝图灯牌后可点歌
- 下一首
  - 大航海，房管，主播可以下一首
  - 下一首不能跳过大航海的点歌
